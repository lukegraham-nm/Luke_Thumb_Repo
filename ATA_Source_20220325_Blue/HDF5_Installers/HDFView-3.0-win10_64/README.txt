HDFView 3.0.0 with hdf4-java:hdf5-java versions 4.2.14;1.10.3
------------------------------------------------------------------------------

This directory contains the binary (release) distribution of
HDFView 3.0 that was compiled on:
      Windows win-amd64

with:  Java JDK 1.8.0_162

It was built with the following options:
   -- STATIC HDF 4.2.14
   -- STATIC HDF5 1.10.3

===========================================================================
Note: By default HDFView runs on the included Java JRE 1.8.
     This can be modified by changing JAVABIN in HDFView\3.0.0\hdfview.bat
===========================================================================

The contents of this directory are:

   COPYING                 - Copyright notice
   README.txt              - This file
   HDFView-3.0.msi       - HDFView Installer

Running
===========================================================================
To install HDFView for Windows:

1. Execute Execute HDFView-3.0.msi
2. Follow prompts
3. Execute HDFView\3.0.0\hdfview.bat to run HDFView
===========================================================================
